# Паралельні обчислення: Алгоритм пошуку регресійного рівняння

## Опис проекту
Курсова робота з дисципліни "Паралельні обчислення" на тему реалізації алгоритму пошуку регресійного рівняння з використанням паралельних обчислень на Java.

## Завдання курсової роботи

1. Розробка послідовного алгоритму
2. Дослідження швидкодії послідовного алгоритму
3. Розробка паралельного алгоритму
4. Тестування коректності обчислень (всі JUnit тести пройдено)
5. Дослідження швидкодії паралельного алгоритму
6. Експериментальне дослідження прискорення
7. Дослідження впливу параметрів паралелізації
8. Висновки про переваги паралельної реалізації

## Компіляція та запуск

### Компіляція проекту:
```bash
mvn clean compile
```

### Запуск тестів:
```bash
mvn test
```

### Запуск програми:
```bash
mvn exec:java -Dexec.mainClass="ua.coursework.regression.Main"
```
